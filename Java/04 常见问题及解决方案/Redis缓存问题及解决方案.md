# Redis缓存三大问题及解决方案

1. **缓存穿透**
   - **定义**：访问数据库和缓存中不存在的值，导致每次请求都需要查询数据库。
   - **解决方案**：
     - 缓存空数据
     - 布隆过滤器
2. **缓存雪崩**
   - **定义**：缓存的可靠性问题，导致无法正常使用，导致大量请求到数据库
   - **解决方案**：
     - 使用缓存集群，保证缓存高可用
     - 防雪崩工具：Hystrix熔断、限流、降级手段进行保护
3. **缓存击穿**
   - **定义**：热点数据集中失效
   - **解决方案**：
     - 互斥锁：只允许一个线程重建缓存，其他线程等待建立完毕以后访问缓存
     - 过期时间随机，不要同时失效，或者永不过期
